# components/BuildingInfo.tsx.md

## 개요
건물 외관, 플로어 플랜 선택, 시설 디렉터리를 보여주는 상세 뷰입니다.

## 동작
- 디렉터리 행을 층별로 그룹화하고, 지하 → 숫자/사전 순으로 정렬합니다.
- 건물이 바뀔 때 `selectedFloorPlanLevel`을 첫 라벨로 초기화합니다.
- 선택된 층의 플로어 플랜 이미지를 보여주고, 없으면 플레이스홀더 메시지를 표시합니다.
- 선택된 층 데이터가 있을 때 디렉터리를 필터링하며, 없을 때 알림을 띄웁니다.

## 렌더
이미지와 오른쪽 컬럼(층 버튼, 플랜 이미지, 디렉터리 리스트) 그리드 구성. Tailwind 클래스 사용. 일부 한국어 문자열이 깨져 있습니다.
